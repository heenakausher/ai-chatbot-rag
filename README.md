# AI Chatbot RAG

A Retrieval-Augmented Generation (RAG) chatbot built with:

- Streamlit
- LangChain
- Groq LLM
- HuggingFace Embeddings
- Chroma Vector Database

## Features

- Basic Chatbot (Phase 1)
- Groq Powered Chatbot (Phase 2)
- PDF RAG Chatbot (Phase 3)

## Installation

```bash
pip install -r requirements.txt
```

## Run

Phase 1

```bash
streamlit run phase1.py
```

Phase 2

```bash
streamlit run phase2.py
```

Phase 3

```bash
streamlit run phase3.py
```
